Verus Tracker

Pick a calendar week; each week is stored separately in your browser. Monday–Saturday scores and VS Total (sum) apply to that week only. Import scores from screenshots or .mp4 screen recordings with OCR.

  Player Del
Import expects columns named like Player and Monday … Saturday. VS Total is computed as the sum of the week (it is used only as a fallback when day cells are blank). Screenshot or screen recording import reads commander names and point totals (OCR). Pick which day column to fill, then choose image(s) and/or an .mp4 (e.g. Recording 2026-03-26 … .mp4 from Game Bar / OBS / similar). Videos skip the first second (often black or UI chrome), then sample about once per second and merge the best score per commander. Names match existing players when possible; otherwise new rows are added.
