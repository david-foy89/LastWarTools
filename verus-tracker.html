Versus Tracker

Pick a calendar week; each week is stored separately in your browser. Monday–Saturday scores and VS Total (sum) use Desert Storm server-day timing: EST + 2 hours, midnight reset. Import scores from screenshots or .mp4 screen recordings with OCR.

  Player Del
Import names (CSV/Excel) is for alliance rosters: a column headed Name, Player, Commander, etc. (e.g. a single Name column on an “Alliance Members” sheet). Check Replace table to wipe this week’s rows and load only that list. Without it, names are appended and duplicates are skipped. Screenshot / video OCR reads the leaderboard and fills the selected day. Keep Match roster only checked (default) so OCR only updates players already in the table—garbled OCR text won’t create extra rows. Uncheck to allow OCR to add new rows. VS Total is the sum of Mon–Sat (or the imported total when day cells are blank).
