Versus Tracker

Pick a calendar week; each week is stored separately in your browser. Monday–Saturday scores and VS Total (sum) use Desert Storm server-day timing: EST + 2 hours, midnight reset. Import scores from screenshots or .mp4 screen recordings with OCR.

Paste leaderboard text

Paste lines that look like the in-game list (rank, name, score). One row per line is fine; tab- or comma-separated also works. This skips image OCR—use it when screenshots read poorly.

Import names (CSV/Excel) is for alliance rosters: a column headed Name, Player, Commander, etc. (e.g. a single Name column on an “Alliance Members” sheet). Check Replace table to wipe this week’s rows and load only that list. Without it, names are appended and duplicates are skipped. Screenshot / video OCR reads the leaderboard and fills the selected day (full-screen PNG captures at native resolution work best; in-game OCR is approximate—verify scores). Recordings are sampled every second (up to five minutes); tall portrait and wide landscape clips also OCR a bottom crop where leaderboards often sit. All sampled frames are merged so partial reads can combine. Pick the narrowest language set that covers your roster (default is Korean + English). Paste leaderboard text skips image OCR entirely—paste from a spreadsheet or phone OCR when screenshots read badly. Keep Match roster only checked (default) so OCR only updates players already in the table—garbled OCR text won’t create extra rows. Uncheck to allow OCR to add new rows. When the leaderboard shows a rank number (e.g. 1. 12)), scores map by that rank to your roster in row list order (the order rows are stored— same as import / “Add player” order, top to bottom on the screenshot). Reorder rows in the table UI if your list order should match the game before using rank OCR. When OCR also reads a commander name, the score is applied to the roster row that matches that name if rank and name disagree (so wrong rank or row order does not move scores onto the wrong player when the name is recognizable). VS Total is the sum of Mon–Sat (or the imported total when day cells are blank). Player cells are normal text fields—edit anytime. OCR also writes a detected name into a row when that row’s name is still empty (e.g. rank-only fill).
